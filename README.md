# 🍔 Hamburgueria API

API REST para gestão de pedidos e cardápio de uma hamburgueria, desenvolvida como projeto prático para o curso de **Análise e Desenvolvimento de Sistemas**.

## 🩰Tecnologias Utilizadas
- **Node.js** & **Express**
- **Sequelize ORM** (MySQL)
- **Arquitetura MVC**

## 🛠️ Implementações Técnicas
- **Modelagem de Dados:** Associações complexas entre tabelas (Pedidos, Entregas, Produtos e Avaliações).
- **Eager Loading:** Otimização de consultas ao banco de dados utilizando `include` para performance.
- **Associações:** Implementação de relacionamentos 1:1 e 1:N.

## 📌 Funcionalidades
- Cadastro e listagem de produtos por categoria.
- Registro de pedidos com vínculo automático de entregas.
- Sistema de feedback (Avaliações) integrado ao histórico de pedidos.
